# Play Store Screenshot Checklist

To submit the MVP to the Play Store, we need to capture the following screenshots:

- [ ] Login screen
- [ ] Workouts list
- [ ] Add workout
- [ ] Edit workout
- [ ] Start workout
- [ ] Workout session screen
- [ ] How-to videos screen
- [ ] Settings/profile (if useful)

**Note:** Ensure all screenshots are from the actual app running on a device or emulator. Do not generate fake screenshots.
