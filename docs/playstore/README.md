# Play Store Documentation

This directory contains documents necessary for the Play Store submission of GainzAssist.

## Privacy Policy (`privacy-policy.md`)

Before submitting GainzAssist to the Google Play Store, please note the following:

- **Public Hosting:** The privacy policy must be hosted publicly on a website (e.g., GitHub Pages, a personal website, or a dedicated privacy policy hosting service). The URL to this publicly hosted policy must be linked in your Google Play Console under the App Content section.
- **Replace Placeholder:** Be sure to replace the `[YOUR_SUPPORT_EMAIL]` placeholder in the privacy policy with your actual support or contact email address before publishing it.
- **Keep it Updated:** You must update the privacy policy if the app's features change, specifically regarding data collection or third-party integrations (e.g., changes to Facebook Login, Firebase Analytics, or YouTube API usage).
